# ClipVault

ClipVault is a full-stack cross-device clipboard and note storage tool designed for modern efficiency.

## 🚀 Features

- **Cross-Device Sync**: Access your notes and clipboard from any device.
- **Rich Content**: Support for text, markdown, and image uploads.
- **Organization**: Pin important notes and categorize with tags.
- **Modern UI**: Dark-themed, responsive, and buttery-smooth design built with TailwindCSS and Framer Motion.
- **Secure**: Industrial-grade security with JWT authentication and bcrypt.

## 🛠 Tech Stack

- **Frontend**: React, Vite, TailwindCSS, Framer Motion, Lucide Icons.
- **Backend**: Node.js, Express.js, MongoDB, Mongoose, Multer.
- **Auth**: JWT, bcryptjs.

## 📦 Getting Started

For detailed setup instructions, please refer to the `setup_instructions.md` in the artifacts directory or follow the quick start below:

### Backend
```bash
cd backend
npm install
npm run dev
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```
